Project Title: Basic Weather Application

Description:
This project is a simple Python-based weather application that uses the OpenWeatherMap API to fetch real-time weather data.
The user enters a city name, and the program displays the current temperature in Celsius along with a brief weather description.
It also handles invalid city names and provides user-friendly output.

Technologies Used:
- Python 3 (Programming Language)

How to Run:
1. Install Python
2. Open command prompt
3. Run: python weather.py
