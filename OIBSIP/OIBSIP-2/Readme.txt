Project Title: RANDOM PASSWORD GENERATOR

Description:
The Random Password Generator is a Python program that creates secure, random passwords based on user-defined criteria. Users can choose the length of the password and whether to include letters, numbers, and symbols. It helps generate passwords that are safe and easy to use.

Technologies Used:
- Python 3 (Programming Language)

How to Run:
1. Install Python
2. Open command prompt
3. Run: python password.py

Features:
=> Generate passwords of any length.
=> Option to include or exclude letters, numbers, and symbols.
=> Handles invalid inputs gracefully.
